# Repository AGENTS

## 工作日志规则（默认启用）

- 每次与用户完成一轮有效协作后，必须记录工作日志。
- 日志目录固定为：`docs/worklog/`
- 日志文件按日期命名：`docs/worklog/YYYY-MM-DD.md`
- 索引文件：`docs/worklog/INDEX.md`（新日期文件创建时需补索引）

## 每条日志至少包含

- `对话目标`
- `执行内容`
- `修改文件`（无修改则写“无”）
- `验证结果`（无验证则写“未执行”）
- `备注`（可选）

## 例外

- 用户明确要求“本次不记录日志”时，可跳过一次。
- 严禁写入密钥、令牌、账号密码等敏感信息。
